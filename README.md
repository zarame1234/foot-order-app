# Food Order App（商品管理・注文アプリ）

## 概要
学習目的でSpring Bootを用いて作成した商品管理・注文アプリケーションです。
商品一覧から商品を選択し、カートへの追加、注文確認、注文確定、注文履歴の確認まで、一連の注文フローを実装しています。

---

## 使用技術
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven
- Bootstrap 5

---

## 主な機能

### 商品機能
- 商品一覧表示
- 商品詳細表示
  
### カート機能
- 商品をカートへ追加
- カート内商品の数量変更
- カート内商品の削除
- カート内商品の一括削除

### 注文機能
- 注文確認画面
- 合計金額の表示
- 注文確定
- 注文履歴表示

### 入力チェック
- 商品数量のバリデーション
- カート更新時のバリデーション
- エラーメッセージ表示

### 例外処理
- 存在しない商品へアクセスした場合のエラー画面表示
