# Food Order App（商品管理・注文アプリ）

## 概要
学習目的でSpring Bootを用いて作成した商品管理・注文アプリケーションです。
商品一覧表示・商品登録機能を実装し、MySQLと連携してデータを永続化しています。  
今後、注文機能や編集・削除機能を順次追加予定です。

---

## 使用技術
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven

---

## 主な機能

### 商品機能
- 商品一覧表示
- 商品詳細表示
- カート追加
- MySQLへのデータ保存
